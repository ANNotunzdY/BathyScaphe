ログを削除する

既得スレッドのログを削除するには、そのスレッドを開くか、またはスレッド一覧で選択します。次に、「スレッド」＞「ログを削除...」と選択します。

  • スレッドにまだ書き込んでいない内容がそのスレッドの書き込みウインドウにある（下書き）場合、下書きも同時に削除されます。
  • 削除されたスレッドのログファイルは、ゴミ箱に移動されます。ログファイルを完全に削除するには、ゴミ箱を空にする必要があります。
  • 「お気に入り」に登録されていたスレッドのログを削除した場合、スレッドは自動的に「お気に入り」から削除されます。

dat 落ちしたために、スレッド一覧から開けないスレッドのログを削除する方法については、dat 落ちしたスレッドのログを削除する を参照してください。

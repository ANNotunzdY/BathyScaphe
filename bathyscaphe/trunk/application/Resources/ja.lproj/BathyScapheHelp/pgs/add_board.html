掲示板を追加する

掲示板リストに掲示板を追加するには、以下のようにします（スマート掲示板を追加したい場合は、かわりに「スマート掲示板を作成する」を参照してください）。

  1. 「掲示板」＞「掲示板を追加...」と選ぶか、または掲示板リストの左下にある追加（＋）ボタンをクリックします。
  2. リストからカテゴリや掲示板を選択して「追加」をクリックします。検索フィールドを利用すると、すばやくリストから探すことができます。
  3. リストに掲示板が見つからない場合は、直接掲示板の名前と URL を入力し、「追加」をクリックします。

ヒント

  • 最近新設された２ちゃんねるの掲示板がリストに見つからない場合は、まず掲示板リストを同期してみます。
    ただし、新設直後の場合は同期してもなお、リストに載っていないこともあります。これは異常ではありません。掲示板リストの提供元の更新を待つか、必要であれば直接掲示板の名前と URL を入力して追加します。
  • livedoorしたらばや、小規模な互換掲示板の場合は、リストに載っていないことが多いため、直接入力する必要があるでしょう。
  • 掲示板の URL は「http://」で始まり、「/」で終わっていなければなりません。末尾に「index.html」などがついている場合は、それを削ったものを入力します。

重要：すでに存在するものと同じ名前の掲示板を追加することはできません。

内容を検索する

BathyScaphe の検索機能を使って、表示中のスレッドからレス、名前、ID などを検索できます。検索結果を抽出（一覧表示）することもできます。

  1. まず、検索したいスレッドを表示します。
  2. 「編集」＞「検索」＞「検索...」と選びます。
  3. 適切なオプションを選択して検索を実行します。
  • 「抽出」をクリックすると、検索語句を含むすべてのレスがポップアップで一覧表示されます。
    ヒント：特定の ID を抽出したいときは、単に ID をマウスでクリックしたまま少し待つだけで抽出できます。
  • 「ハイライト」をクリックすると、見つかった検索語句がすべてハイライト（強調表示）されます。
  • 「リンクのみ検索」オプションは、「抽出」と「ハイライト」では無効です。

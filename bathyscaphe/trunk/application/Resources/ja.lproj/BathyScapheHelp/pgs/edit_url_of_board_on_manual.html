掲示板の URL を手動で編集する

掲示板リストに登録された掲示板の URL を、手動で編集することができます。サーバ移転直後や２ちゃんねるのトラブル発生時、または２ちゃんねる以外の掲示板である等の理由で、自動的な URL の更新が利用できないときにこの方法を利用します。

掲示板の URL を手動で編集するには：

  1. 掲示板リストで、編集したい掲示板を選択します。
  2. 「掲示板」＞「編集…」と選択します。
  3. 新しい URL を入力し、「OK」をクリックします。

次にスレッド一覧を更新したり、スレッドを開くときから、新しい URL が使用されるようになります。

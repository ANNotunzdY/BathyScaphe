名前欄、メール欄、および Be の状態をあらかじめ設定する

いつも同じ名前（コテハン）で書き込みたい、あるいはいつもメール欄に「sage」を入力したいときなど、名前欄とメール欄の内容をあらかじめ設定しておくことが可能です。さらに、掲示板ごとに Be をデフォルトで有効にするかどうかも設定できます。

これらの設定は、「すべての掲示板」または「特定の掲示板」ごとに設定できます。

  • すべての掲示板で名前・メール欄・Be の状態をあらかじめ設定するには、「書き込み」環境設定の「名前欄とメール欄」で設定します。詳細は「関連項目」のリンクをクリックしてください。
  • 掲示板ごとに名前欄・メール欄の内容を設定するには、「掲示板」＞「掲示板オプションを表示」と選択します。次に「書き込み」をクリックし、名前欄とメール欄を入力します。
    参考：掲示板ごとの設定は、すべての掲示板に対する設定よりも優先されます。
  • 掲示板ごとに Be をデフォルトで有効にするかどうか設定するには、「掲示板」＞「掲示板オプションを表示」と選択します。次に「書き込み」をクリックし、「Be を有効にする」チェックボックスにチェックを入れます（あるいは、外します）。
    参考：常に Be ログインが必要な掲示板や、２ちゃんねる以外の掲示板では、この設定は無視されます。

これらの設定内容は、書き込みウインドウに「あらかじめ入力される」だけであることに注意してください。名前欄・メール欄の内容や、Be の有効／無効は、必要に応じて実際の書き込み時にも変更できます。

関連項目

  • 「名前欄とメール欄」を設定する
  • 掲示板オプションを設定する
  • スレッドに書き込む
  • Be を有効にして書き込む

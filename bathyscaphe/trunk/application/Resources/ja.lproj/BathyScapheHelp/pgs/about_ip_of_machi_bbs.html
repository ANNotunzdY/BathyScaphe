まちBBSのスレッドでIPが表示されない

まちBBSのスレッドで、各レスの投稿元リモートホスト情報（IP）が表示されないのは、異常ではありません。このバージョンの BathyScaphe の正常な仕様です。

このバージョンの BathyScaphe では、各レスの IP を取得・表示しません。これは、まちBBSからデータを取得する方法を変更したためです（変更後の方法では、まちBBS側が IP のデータを提供していません）。

注意：以前のバージョンの BathyScaphe で取得した分のレスについては、IP のデータがログファイルに記録されているため、このバージョンの BathyScaphe でも引き続き IP が表示されます。

スレッドを立てる

BathyScaphe から直接スレッドを立てることができます。

重要：２ちゃんねる以外の掲示板では、スレッドを立てられないことがあります。

スレッドを立てるには：

  1. ブラウザウインドウで、スレッドを立てたい掲示板を表示します。
  2. 「掲示板」＞「新規スレッドを作成...」と選択します。
    ヒント：ブラウザウインドウのツールバーをカスタマイズすると、ツールバーから新規スレッド作成ウインドウを開くこともできるようになります。
  3. 新規スレッド作成ウインドウは、書き込みウインドウとほぼ同様ですが、スレッドタイトルを入力するフィールドが追加されています。
  4. 必要な内容を入力したら、「送信」をクリックします。

BathyScaphe はスレッドを立てた後、約1秒後に自動的にスレッド一覧を更新し、今立てたスレッドを探します。見つかれば、そのスレッドを自動的に開きます。

重要： 掲示板によっては、反映に時間がかかるため、立てたスレッドを見つけられない場合があります。また、同じタイトルのスレッドが複数ある場合、間違ったスレッドを開いてしまう場合があります。自動的に開いたスレッドの内容については、必ずご自身でご確認ください。

関連項目

  • ツールバーをカスタマイズする
  • スレッドに書き込む
  • 「アカウント」を設定する
  • テンプレートを挿入する

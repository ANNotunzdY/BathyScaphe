レスの属性を設定する

レスには次のような属性を設定することができます。

  • 迷惑レス
  • ローカルあぼーん
  • 透明あぼーん
  • AA
  • ブックマーク

レスに属性を設定するには、次のようにします。

  1. レス番号をクリックします（control キーを押す必要はありません）。
  2. メニューから設定したい属性を選びます。

連続した複数のレスに一度に同じ属性を設定するには、次のような方法もあります。

  1. スレッドのテキストを、属性を設定したいレスに範囲がかかるように選択します。
  2. 選択したテキスト上で control キーを押しながらクリックして、メニューから設定したい属性を選びます。

また、「スレッド」＞「レスの属性」サブメニューで、その属性が設定されたレスを一度に見ることができます。

設定された属性を解除する方法については、レスの属性を解除する を参照してください。

各属性の意味については

  • 迷惑レスは迷惑レスフィルタを使う
  • ローカルあぼーん、透明あぼーんは任意のレスをあぼーんする
  • AAはAA フォントで表示する
  • ブックマークは特定のレスをブックマークする

を参照してください。

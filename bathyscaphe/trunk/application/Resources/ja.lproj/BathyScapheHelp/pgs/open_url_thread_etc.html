スレッドや掲示板の URL を開く

スレッドや掲示板の URL から直接スレッドを開いたり、スレッド一覧を表示させることができます。

URL から直接スレッド／掲示板を開くには：

  1. 「ファイル」＞「URL を開く...」と選びます。
  2. URL を入力し、「OK」をクリックします。
  3. BathyScaphe がその URL をスレッドまたは掲示板のものと解釈できた場合、スレッドまたはスレッド一覧が表示されます。そうでなければ、かわりに Web ブラウザで開くかどうか尋ねるダイアログが表示されます。

ほかのアプリケーションから URL を BathyScaphe で開くには、URL のテキストを選択して、「サービス」メニューから「URL を BathyScaphe で開く」を選びます。
